# [GitHub/PBougie](https://pbougie.github.io/)


## Overview

Static website generator for [pbougie.github.io](https://pbougie.github.io/).
Built with [Hugo](https://gohugo.io/).


## CLI

- Install Hugo: `brew install hugo`
- Build website: `hugo`
- Watch website: `hugo --watch`
- Run local development server: `hugo server`
- Deploy website to production server: `./deploy.sh`


## Deployment

The website is generated into a separate repository (`pbougie.github.io`) for [GitHub Pages](https://pages.github.com/).
