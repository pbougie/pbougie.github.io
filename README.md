# [pbougie.github.io](https://pbougie.github.io/)


## Overview

Static website generator for [pbougie.github.io](https://pbougie.github.io/).
Built with [Hugo](https://gohugo.io/).


## CLI

- Install Hugo: `brew install hugo`
- Build website: `make build`
- Watch website: `make watch`
- Deploy website to GitHub: `make deploy` (follow instructions)


## Deployment

The website is generated into folder `/docs` for [GitHub Pages](https://pages.github.com/).
